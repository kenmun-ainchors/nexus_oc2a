# Groom vs Plan — Distinction

## Groom
- Analyze ticket.
- Refine scope.
- Surface clarifications, assumptions, risks, open questions.
- Confirm decisions with human.
- Output: **groomed brief** (scope + decisions), not an execution plan.

## CREST Step 1 — Plan
- Input: groomed brief.
- Output: execution plan (atoms, owners, schedule, verify steps, rollback, evidence standards).

## Rule
- Groom first. Then CREST Plan. Then dispatch Execute.
- Never conflate groom atoms with CREST plan atoms.

## Lessons Learned (Day 17 morning)

- **L-022 (reinforced):** Heartbeat must NEVER run EOD close regardless of state flags. Time check is not enough — the task must be owned exclusively by its cron. No fallback, no "null check" workaround.
- **L-023 (new):** Isolated cron sessions cannot write to `/tmp`. Always use `workspace/tmp/` for cron-generated intermediate files.
- **L-024 (new):** Journal format validation must be automated. Human review of journal structure cannot be relied on to catch corruption promptly. auto-heal CHECK 14C now handles this.
- **L-025 (new):** Parallel sessions (e.g., email-triggered work, sub-agent deliveries) are invisible to main session journal reconstruction. Always flag parallel session work explicitly for journal capture.

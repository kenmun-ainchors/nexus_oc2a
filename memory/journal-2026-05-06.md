- `HEARTBEAT.md` updated: routing audit added as periodic check
- PVT expanded: **Check 11** added — validates that all Angie-destined message delivery uses accountId:aria
- auto-heal expanded: **Check 14B** added — detects bot routing mismatches on outbound Telegram deliveries

**Outcome:** Bug fixed. Angie will receive all messages through the Aria bot as intended. Two systemic hardening additions (PVT-11, auto-heal-14B) prevent recurrence. Routing audit script available for manual checks.

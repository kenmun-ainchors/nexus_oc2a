## Rules Established Today

### Pre-Risky-Op Checkpoint (NON-NEGOTIABLE)
Before any openclaw update/restart/major config change:
1. Flush in-progress work to MEMORY.md + daily notes
2. Write decisions to decisions.md
3. Update Notion sprint status
4. Git commit workspace
5. Clear stale plugin-runtime-deps: `rm -rf ~/.openclaw/plugin-runtime-deps/openclaw-unknown-*`
6. Confirm to Ken: "Checkpoint saved. Safe to proceed."
Post-op: run PVT (pvt.sh). All 9/9 must pass before resuming.

Full platform verification: gateway, Ollama, health checks, memory, doctor, tasks, secrets, Telegram. Now formalised as PVT (Post Verification Test) via scripts/pvt.sh — 9/9 checks.

## Day 3 Lessons

1. **Files are the recovery mechanism.** Gateway crashed, context was fully restored from files in 10 minutes. Invest in files, not in trusting memory or session state.
2. **Gemma4 = background only, forever.** 26B model cold-loading on every interactive message is a UX killer. Never route interactive traffic to it.
3. **Config drift is the stealth enemy.** Two Opus drift incidents in one day — both from config overrides that survived restarts. Critical config baseline + nightly check is mandatory, not optional.
4. **Single-bot routing with two agents is fragile.** Stale session state caused days of routing confusion. Hard-wired two bots → problem solved instantly. Sometimes the architectural fix is cleaner than the config fix.
5. **Ticket-first discipline catches scope creep early.** TKT-0001 raised against the ITIL work itself — the system bootstrapped its own discipline.

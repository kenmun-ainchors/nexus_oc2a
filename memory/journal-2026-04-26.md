#### US12: Async Execution Model

Built the full async execution protocol for long-running and multi-step tasks.

Components delivered:
- `scripts/task-create.sh` — creates TASK file at `handoff/TASK-{YYYYMMDD}-{NNN}.md`
- `scripts/task-checkpoint.sh` — writes checkpoint after every step
- `scripts/task-watchdog.sh` — runs every 30 min, alerts Ken if task stalled >30 min
- `scripts/task-complete.sh` — marks task done, triggers summary
- Resume protocol: read TASK file → find last checkpoint → continue. Never restart from scratch.
- Max 2 retries per step, then mark `blocked` and escalate to Ken.
- Notification rule: Notify Ken at start, 50%, done or blocked. Never on every step.

Full protocol: `~/Documents/AInchors/Operations/AsyncExecution.md`
SOUL.md updated with async execution model section.

### 20:52 — INC-20260426-003: Gateway Restart ENOTEMPTY Crash Loop

**Type:** Unplanned outage — crash loop  
**Duration:** ~116 min  
**Root cause:** `openclaw gateway restart` triggered a crash loop. The `~/.openclaw/plugin-runtime-deps/` directory contained stale `openclaw-unknown-*` directories from a previous failed update (INC-002). On restart, the runtime attempted to write to these directories and hit an `ENOTEMPTY` error, causing a restart loop.

**Impact:**
- ~116 min recovery time (longest outage of the day)
- Gateway in crash loop — no API responses
- Ken had to monitor and wait for manual intervention
- Eventually resolved by: `rm -rf ~/.openclaw/plugin-runtime-deps/openclaw-unknown-*` then clean restart

**Fix:**
- Pre-risky-op checkpoint rule **extended** to include `plugin-runtime-deps` cleanup step
- Step added: `rm -rf ~/.openclaw/plugin-runtime-deps/openclaw-unknown-* 2>/dev/null; ls ~/.openclaw/plugin-runtime-deps/` — confirm only one versioned dir remains
- Decision logged in `memory/shared/decisions.md`

Ken defined "shakedown test" as a structured post-incident validation: run the system through its key functions to confirm recovery before declaring the incident resolved. Adopted as standard practice.
